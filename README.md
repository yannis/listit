# nos_listes

A shopping list sharing app for family members

## stack

- Rails
  - authentication: Devise
  - authorization: CanCanCan
  - Turbolinks
  - Postgres
  - ActiveAdmin for admin backend
- View components
- Tailwind css
- Stimulus and Stimulus reflex
- PWA
